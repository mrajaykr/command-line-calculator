# command-line-calculator

## Objective
 Create a command-line calculator supporting basic operations.

## Discription
A simple Command-Line Calculator built in Python that performs basic arithmetic operations (addition, subtraction, multiplication, and division) using a menu-driven interface.

## Tools Used
- Python, VS Code / any text editor, terminal.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Exit Option

## How to Run

```bash
python calculator.py
```

## Sample Output

```text
===== CALCULATOR =====
1. Addition (+)
2. Subtraction (-)
3. Multiplication (*)
4. Division (/)
5. Exit

Enter your choice (1-5): 1
Enter first number: 7
Enter second number: 7

Result = 7.0
```
